## DTO定义

#### Goods

```js
{
    "URL":"String",    //商品的url
    "origin":"String", //商品的来源,如天猫
    "name":"String",   //商品名称
    "price":2.31,      //商品价格
    "store":"String"   //商品的店铺名称
}
```

#### Comment

```js
{
    "userId":"String",     //评论者ID
    "content":"String",    //评论内容
    "GoodsId":"String",    //被评论的商品ID
    "datetime":yyyy-MM-dd hh:mm:ss    //评论时间
}
```
